# 🌿 Nature Landing Page

A modern, responsive, and visually appealing landing page built using **HTML5** and **CSS3**. This project showcases a clean user interface with a nature-inspired theme, designed to promote environmental awareness and outdoor exploration.

## ✨ Features

* Responsive Navigation Bar
* Attractive Hero Section
* Trusted Companies Section
* Clean and Modern UI
* Nature-themed Design
* Responsive Layout using Flexbox
* Call-to-Action Button
* Optimized Images

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* Google Fonts
* Visual Studio Code

## 🚀 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/shaivya-mehrotra/landing_page.git
```

2. Open the project folder.

3. Launch `landing.html` in your preferred web browser.
